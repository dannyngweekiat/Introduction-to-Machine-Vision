# Introduction to Machine Vision

Welcome! This is a beginner-friendly introduction to **computer vision**: teaching computers how to look at and learn from pictures.

The eight notebooks in this repository are for secondary school students. You do not need previous programming or AI experience. Work through them in order, experiment with the code, and focus on explaining what you notice.

## Start learning

The easiest way to use the notebooks is with **Google Colab**, a free website that runs Python in your browser.

1. Choose a lesson below and select **Open in Colab**.
2. In Colab, choose **Runtime > Run all**, or run one cell at a time using the play button.
3. Read the explanations, change the challenge code, and see what happens.
4. Save your own copy with **File > Save a copy in Drive** before making larger changes.

You will need a Google account to save work in Colab. Lessons 5 and 6, plus Lesson 8 Projects C and D, download pre-trained AI models, so your device needs an internet connection for those activities.

## Use AI thoughtfully

- Use the sample images and datasets in the notebooks, or images you have permission to use. Do not upload private or personal photographs.
- An AI prediction is a helpful guess, not a fact. Test it with different examples and look for mistakes or unfair results.
- Ask a teacher or trusted adult before sharing your work, images, or personal information online.
